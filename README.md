## ✨ About The Project
Welcome to the official web application for **Gurujii Group of Real Estates**. Based in Madurai, Gurujii Group has over two decades of excellence in strategic land acquisitions, dealing in premium DTCP-approved plots, luxury villas, and farm lands. 

This website is a highly optimized, fully responsive Single Page Application (SPA) designed to reflect the luxury and reliability of the Gurujii brand. It features a custom "Liquid Glass" design system, cinematic AI-generated imagery, and smooth scroll animations.

---

## 🚀 Key Features

* 🌟 **Liquid Glassmorphism UI:** Stunning glass-panel overlays, smooth gradients, and elegant serif typography.
* 📱 **100% Fully Responsive:** Flawless scaling from massive 4K desktop monitors down to mobile iPhones.
* ⚡ **Lightning Fast:** Built on React + Vite, delivering instant page transitions and highly compressed assets.
* 🎥 **Cinematic Animations:** Uses `framer-motion` and `GSAP` to deliver premium scroll-triggered reveals and micro-interactions.
* 🔍 **Ultimate SEO:** Contains 100+ hyper-targeted keywords, robust meta descriptions, Canonical Links, and Open Graph tags for WhatsApp/Facebook sharing.
* 💬 **WhatsApp Integration:** Direct-to-WhatsApp contact forms that dynamically generate customized inquiry messages.

---

## 🛠️ Technology Stack

| Technology | Purpose |
| ------ | ------ |
| **React (v18)** | Core JavaScript Framework |
| **Vite** | Blazing fast build tool and dev server |
| **Tailwind CSS** | Utility-first CSS styling framework |

| **React Router Dom** | Client-side page navigation |

## 📂 Project Structure

```bash
├── public/                 # Static assets (Favicons, Logos, AI Generated Project Images)
├── src/
│   ├── components/         # Reusable UI components (Navbar, Footer, ProjectCard)
│   ├── pages/              # Main routes (Home, About, Projects, Services, Contact)
│   ├── index.css           # Global Tailwind CSS configurations and glass-panel classes
│   └── main.jsx            # React root and router provider
├── tailwind.config.js      # Custom theme colors (Primary: #800000, Secondary: #B8860B)
├── vercel.json             # Vercel deployment configurations and route rewrites
└── index.html              # HTML entry point loaded with SEO tags
```

---

## 🏃‍♂️ Running Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

---

## 🌐 Deployment (Vercel)

This application is fully optimized to be hosted on Vercel. A `vercel.json` file is included in the root directory to automatically handle client-side routing rewrites, preventing `404 Not Found` errors when refreshing sub-pages like `/about`.
